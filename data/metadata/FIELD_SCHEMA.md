# Field Schema

## `patent_texts.csv`

| Field | Description |
| --- | --- |
| `patent_title` | Patent title from a public patent page |
| `patent_abstract` | Patent abstract from a public patent page |

## Topic assignment files

| Field | Description |
| --- | --- |
| `topic_id` | Topic index from the NMF run |
| `topic_name` | Human-readable topic name from the source analysis |
| `topic_keywords` | Top terms associated with the topic |
| `patent_title` | Patent title |
| `patent_abstract` | Patent abstract |

CETopic files use `cetopic_subtopic_id`, `cetopic_subtopic_name`, and `manual_category` for the subtopic layer.
